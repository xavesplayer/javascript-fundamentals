Operadores de atribuição 


(Quando uma var recebe ela mesma)
var n = 3
n = n + 4   ou   n += 4 
n = n - 5   ou   n -= 5
n = n * 4   ou   n *= 4
n = n / 2   ou   n /= 2
n = n ** 2  ou   n **= 2
n = n % 5   ou   n %= 5

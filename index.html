Precedência de operadores

  • 1º ()
  • 2º **
  • 3º * / %
  • 4º + -
  • Relacionais: não possuem ordem definida
  • Ordem lógica: ! → && → ||

Operadores de Atribuição


var n = 3
n = n + 4   // ou n += 4
n = n - 5   // ou n -= 5
n = n * 4   // ou n *= 4
n = n / 2   // ou n /= 2
n = n ** 2  // ou n **= 2
n = n % 5   // ou n %= 5

Operadores de Incremento


var x = 5

++x   // pré-incremento
x++   // pós-incremento

--x
x--

Operadores Relacionais

  • 5 > 2 → true
  • 7 < 4 → false
  • 8 >= 8 → true
  • 9 <= 7 → false
  • 4 != 4 → false
  • 5 == '5' → true
  • 5 === '5' → false
  • 5 !== '5' → true
O resultado sempre será booleano.
Evite usar ==. Prefira === para comparar tipo e valor.

Operadores Lógicos

  • ! → negação
  • && → e
  • || → ou

idade >= 15 && idade <= 17
estado == 'RJ' || estado == 'SP'
salario > 1500 && sexo != 'M'

Operador Ternário


media >= 7 ? "Aprovado" : "Reprovado"
? funciona como if e : como else.

var d = document.querySelector('#msg')

d.style.color = 'blue'
Quando há "Elements" (no plural) retorna vários elementos, por isso usamos índice [0].
innerText mostra só o texto. innerHTML inclui tags internas.

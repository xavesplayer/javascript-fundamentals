Precedência de operadores

  • 1º ()
  • 2º **
  • 3º * / %
  • 4º + -
  • Relacionais: sem ordem definida
  • ! → && → ||

Operadores de Atribuição


var n = 3
n += 4
n -= 5
n *= 4
n /= 2
n **= 2
n %= 5

Operadores de Incremento


var x = 5
++x   // pré
x++   // pós
--x
x--

Operadores Relacionais

  • 5 > 2 → true
  • 7 < 4 → false
  • 8 >= 8 → true
  • 5 == '5' → true
  • 5 === '5' → false

Use === para evitar problemas de tipo.

Operadores Lógicos

  • ! → negação
  • && → e
  • || → ou

idade >= 15 && idade <= 17
estado == 'RJ' || estado == 'SP'

Operador Ternário


média >= 7 ? "Aprovado" : "Reprovado"

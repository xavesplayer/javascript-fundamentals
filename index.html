Precedência de operadores

  • 1º ()
  • 2º **
  • 3º * / %
  • 4º + -
  • Relacionais: não possuem ordem definida
  • Ordem lógica: ! → && → ||

Operadores de Atribuição


var n = 3
n = n + 4   //  n += 4
n = n - 5   //  n -= 5
n = n * 4   //  n *= 4
n = n / 2   //  n /= 2
n = n ** 2  //  n **= 2
n = n % 5   //  n %= 5

Operadores de Incremento


var x = 5

++x
x++

--x
x--
++x incrementa o valor antes de usá-lo. x++ incrementa depois de usá-lo.

Operadores Relacionais

  • 5 > 2 → true
  • 7 < 4 → false
  • 8 >= 8 → true
  • 9 <= 7 → false
  • 4 != 4 → false
  • 5 == '5' → true
  • 5 === '5' → false
  • 5 !== '5' → true
O resultado sempre será booleano.
Evite usar ==. Prefira === para comparar tipo e valor.

Operadores Lógicos

  • ! → negação
  • && → e
  • || → ou

idade >= 15 && idade <= 17
estado == 'RJ' || estado == 'SP'
salario > 1500 && sexo != 'M'

Operador Ternário


media >= 7 ? "Aprovado" : "Reprovado"
? funciona como if e : como else.

Condições

Condições simples e compostas


  if (condição) {
    true
  }
  else if (condição) {
    true
  }
  else {
    false
  }

Condição múltipla


  switch (condição) {
    case valor1:
    // código
    break
    case valor2:
    // código
    break
    default:
    // código
  }

Exemplo de switch/case/default


  var agora = new Date()
  var diaSem = agora.getDay()
console.log(diaSem)
switch (diaSem){
    case 0:
        console.log('Domingo')
        break
    case 1:
        console.log('Segunda-feira')
        break
    case 2:
        console.log('Terça-feira')
        break
    case 3:
        console.log('Quarta-feira')
        break
        case 4:
        console.log('Quinta-feira')
        break
        case 5:
        console.log('Sexta-feira')
        break
        case 6:
        console.log('Sábado')
        break
        default:
        console.log('Dia inválido')
        break
      }
O switch é mais eficiente que if-else para muitas condições. Inclusive para o exemplo acima, onde o if-else seria mais complexo.
O break é necessário (obrigatório) para evitar que o código continue executando os próximos cases.
O switch só aceita números inteiros, strings ou booleanos. Não funciona com intervalos. Fazendo-o ser um pouco mais limitado que o if-else.
